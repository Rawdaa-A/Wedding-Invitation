# Mai & Ahmed Wedding Invitation Website

## 🎉 Overview

A premium, highly interactive wedding invitation website built with React, TypeScript, and Framer Motion. The site features elegant animations, interactive elements, and a luxurious minimalist design inspired by Islamic spirituality and contemporary aesthetics.

## ✨ Key Features

### 1. **Loading Screen**
- Beautiful fade-in animation with "Mai & Ahmed" names
- Soft glow and shimmer effects
- Decorative sparkle animations
- Smooth transition to envelope page

### 2. **Envelope Interaction**
- Realistic envelope image with premium styling
- Hover animations (elevation effect)
- Floating animation
- Click-to-open interaction
- Decorative rotating elements

### 3. **Main Invitation Page**
- **Quranic Verse (Surah Ar-Rum 30:21)** - Styled in elegant serif typography
  - Arabic text with Cormorant Garamond font
  - English translation
  - Spiritual foundation of the invitation
  
- **Wedding Details**
  - Names: Mai & Ahmed
  - Date: 30 June 2026
  - Location: Olivia Hall, Talkha, Armed Forces Club, Nile Corniche
  - Google Maps integration

### 4. **Countdown Timer**
- Real-time countdown to wedding date
- Displays: Days, Hours, Minutes, Seconds
- Animated number updates
- Premium gradient styling

### 5. **Interactive Maze Game**
- Help the groom reach the bride
- Keyboard controls (Arrow keys or WASD)
- Touch button controls for mobile
- Success animation with confetti
- Celebratory message on completion

### 6. **Background Music Toggle**
- Floating music button (bottom-right)
- Play/pause functionality
- Animated icon rotation when playing
- Smooth transitions

### 7. **Responsive Design**
- Mobile-first approach
- Fully responsive across all devices
- Touch-friendly interactive elements
- Optimized performance

## 🎨 Design Philosophy

**Minimalist Luxury with Spiritual Elegance**

- **Color Palette**: Off-white, soft blush, champagne gold, deep charcoal
- **Typography**: 
  - Playfair Display (display/headings)
  - Lora (body text)
  - Cormorant Garamond (Arabic text)
- **Animations**: Smooth, reverent, never overwhelming
- **Layout**: Asymmetric, breathing spaces, generous negative space

## 🛠️ Technology Stack

- **React 19** - UI framework
- **TypeScript** - Type safety
- **Tailwind CSS 4** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons
- **shadcn/ui** - UI components

## 🔧 Customization

All design elements can be easily customized:
- Colors in `client/src/index.css`
- Fonts in `client/index.html`
- Animations in component files
- Content in component JSX
